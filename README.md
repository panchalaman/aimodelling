# AI Modelling

A collection of AI/ML model implementations and experiments.

## Overview

This repository contains various machine learning and deep learning model implementations, including:

- Supervised Learning Models
- Unsupervised Learning Models
- Deep Neural Networks
- Model Evaluation & Comparison
- Feature Engineering Pipelines

## Tech Stack

- Python
- TensorFlow / PyTorch
- scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn

## Getting Started

```bash
git clone https://github.com/panchalaman/aimodelling.git
cd aimodelling
pip install -r requirements.txt
```
